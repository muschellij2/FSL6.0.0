Overview

This is a beta-version, somewhat miscellaneous, set of scripts and hints on how to carry out various network-related analyses.

The starting assumption is that you have a set of N timecourses from S subjects' FMRI datasets. For example, you ran MELODIC group-ICA with a dimensionality of N, and then used dual regression to generate S subject-specific versions of the N timecourses. Alternatively, you might have used a set of template images or ROIs from another study, to feed into the dual regression.

Now you are ready to compute a network matrix for each subject, which in general will be an NxN matrix of connection strengths. The simplest approach would be just to use simple correlation, giving an NxN matrix of correlation coefficients.
